# ✨ Little Wins

### One Percent Better, Every Day.

Little Wins is a wellness-focused web application designed to help users improve their health through small, consistent actions.

Unlike traditional fitness apps, Little Wins focuses on sustainable self-improvement rather than extreme dieting, intense workout programs, or obsessive tracking. The goal is to create a supportive space where users can build healthier habits, track their progress, and celebrate small wins along the way.

---

## 🌷 Features

### 🏠 Dashboard

View your current progress at a glance, including:

* Weight goals
* Workout streaks
* Hydration progress
* Nutrition progress
* Current fitness phase
* Daily motivation

### 🫶 Daily Check-In

Track daily wellness through:

* Mood
* Energy
* Appetite
* Workout completion
* Hydration goals
* Nutrition goals
* Period tracking

### 💧 Water Tracker

Interactive hydration tracker with visual water drops to help users reach their daily water intake goals.

### 🍽 Nutrition Tracker

Track calories and protein intake through a simple food journal designed to support healthy weight gain and balanced nutrition.

### 🏋️ Workout Tracker

Follow structured weekly workout plans focused on:

* Glute and thigh development
* Posture improvement
* Core strength
* Upper body strength
* Wrist and forearm strengthening
* Overall consistency

### 📈 Progress Tracking

Monitor physical progress through:

* Weight tracking
* Weekly measurements
* Progress visualization
* Goal tracking

### 🏆 Achievement System

Unlock badges and milestones by maintaining consistency, meeting goals, and building healthy habits.

### 🌷 Period Comfort Mode

A comfort-focused experience that adapts the app during period days with:

* Supportive messaging
* Comfort routines
* Cozy Rest Days
* Streak protection
* Consistency preservation

### 🌙 Dark Mode

A warm and calming dark theme designed for comfortable nighttime use.

## 🚀 Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
